## **ğŸš€ Getting Started**  
### **Prerequisites**  
- Node.js / Python 3.8+  
- MongoDB / PostgreSQL  
- API keys (Twilio, Google Maps, etc.)  

### **Installation**  
1. Clone the repo:  
   ```bash  
   git clone https://github.com/yourusername/CorruptGuard-Helpline.git  
   ```  
2. Set up backend:  
   ```bash  
   cd server  
   npm install  
   npm start  
   ```  
3. Set up frontend:  
   ```bash  
   cd client  
   npm install  
   npm run dev  
   ```
